Mathematical

A Simple CSS Framework

What is Mathematical?

Mathematical is an extremely simple CSS framework intended for prototyping and designing in the browser. It's also great for beginners. Using multiclass CSS patterns, Mathematical's classes behave in a predictable and intuitive manner. With just a quick glance at the markup, almost anyone familiar with CSS can tell what is happening.

Mathematical is based upon BASSCSS and adds a lot of useful modules that you can add or remove from your project.
